## 概要

トレーニングの記録ができるサービス

https://github.com/masaka88/training-app

## 要求

- Web ブラウザで Web アプリケーションとして利用できる。
- 保存したら Slack に投稿される。
  - slack に投稿する文言だけ作るでも良い。
- ランクが自動的に更新される。

## オプショナル要求

- ある程度過去の記録内容を再利用できるにしたい。
  - TODO: 詳細を検討
- iOS アプリとして利用できる。
- Android アプリとして利用できる。
- macOS アプリとして利用できる。
- Windows アプリとして利用できる。

## MVP

- 記録の文言を書いて Slack に投稿する用の文言を作成してコピペできる。
  - slack へは自分で投稿する。
  - テンプレ
  ```markdown
  _日付_
  _何をしたか_
  _どれくらいやったか_
  _コメント・気づきなど_
  _どこでやったか_
  _〇〇月の運動回数_
  ```
  - 出力例
  ```markdown
  _日付_
  2025/09/15
  _何をしたか_
  プッシュアップ
  シットアップ
  アブローラー
  ケーブルクロス
  エアロバイク 40 分
  _どれくらいやったか_
  1 時間
  _コメント・気づきなど_
  暑かったけど頑張りました!
  _どこでやったか_
  ジム
  _9 月の運動回数_
  3 回目
  ```

## ランクについて
- ランクのポリシー
  ```
  週 4 回以上を 3 ヶ月継続したら、マスターランクに昇格
  週 3 回以上を 1 ヶ月継続したら、ゴールドランクに昇格
  週 2 回以上を 1 ヶ月継続したら、シルバーランクに昇格
  週 1 回以上を 1 ヶ月継続したら、ブロンズランクに昇格
  それ以下の場合は準会員。
  ```
